Chuyển khách hàng và lịch sử chat từ Respond.io / Telegram

Bắt tin nhắn pending từ Telegram

Khi bạn chuyển bot từ Respond.io sang đây, có thể có tin nhắn đến trong khoảng trống chưa được xử lý. Tính năng này gọi getUpdates để bắt tất cả những tin đó về.

⚠ Chỉ lấy được tin nhắn chưa được xử lý (Respond.io chưa consume). Không lấy được tin nhắn cũ đã processed.

1

Upload file Contacts từ Respond.io

Settings → Data Export → Contacts → Export CSV. Wizard sẽ tự nhận dạng format và hướng dẫn các bước tiếp theo.

✓ Respond.io format detected
2

Map kênh Respond.io → Channel CRM

💡 2 kênh dùng chung 1 bot: Chọn cùng một Channel CRM cho cả hai — contacts sẽ được gộp vào đó.
3

Lifecycle stages sẽ được thêm vào CRM

Contacts đã tồn tại sẽ được cập nhật (không xóa)

Format JSON — Contacts + Messages

Tự động nhận dạng format: Unicorn JSON, Respond.io export, hoặc Telethon export. Tất cả đều được parse tự động.

Channel đích (cho contacts mới)

Khi nào dùng script này?

  • Respond.io kết nối bằng tài khoản Telegram cá nhân (không phải bot)
  • Muốn export toàn bộ lịch sử chat từ tài khoản Telegram của bạn
  • Cần lấy danh sách người đã nhắn tin với bạn

⚠ Lưu ý: Script này chỉ đọc lịch sử từ TÀI KHOẢN USER của bạn, không đọc được lịch sử BOT. Nếu dùng BOT, hãy dùng Respond.io Data Export thay thế.

Các bước thực hiện

  1. 1Tải Python: python.org
  2. 2Chạy: pip install telethon
  3. 3Lấy API credentials tại my.telegram.org → API development tools
  4. 4Chạy script bên phải, nhập số điện thoại + OTP
  5. 5Upload file telegram_export.json vào tab JSON ở trên
